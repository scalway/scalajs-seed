To run app
-----------------------------------

```
sbt 
> ~app/fastOptJS
``` 

And next open `index.html` in a browser.

testing app
-----------------------------------

to run tests:

```
sbt
app/test
``` 

you need to install also node.js & node-dom

```bash
#instalacja noda
nvm install 5.0 

#instalacja dom
npm install jsdom
```

more on <https://www.scala-js.org/doc/project/js-environments.html>
